# Plataforma de instituciones educativas

Base de desarrollo para una solución integral orientada a colegios, escuelas, academias e institutos.

## Estado actual
Proyecto en fase de bootstrap arquitectónico y técnico.

## Estructura
- `apps/web` → frontend
- `apps/api` → backend
- `packages/*` → módulos compartidos
- `infrastructure/*` → despliegue y operación
- `docs/` → documentación funcional y técnica
- `scripts/` → automatizaciones del proyecto

## Documentos base
- `ARQUITECTURA-INICIAL.md`
- `BLUEPRINT-EDUCA.md`

## Objetivo inmediato
Levantar una primera base publicable en `educa.hacktrickstore.com` para ver avances reales mientras se construye el sistema.

## Acceso local de desarrollo

- Frontend: `http://localhost:3000`
- Backend: `http://localhost:4100`
- Health API: `http://localhost:4100/api/system/health`
